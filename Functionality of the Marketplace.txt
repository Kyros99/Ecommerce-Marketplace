Laravel Academy Project - Marketplace
Το project αυτό έχει σαν στόχο τη δημιουργία μιας εφαρμογής κατηγορίας marketplace η οποία θα επιτρέπει σε καταναλωτές να βλέπουν τα προϊόντα, να κάνουν ειδικές αναζητήσεις, να τα βάζουν σε καλάθι και κάνουν παραγγελίες. Επιπρόσθετα, θα επιτρέπει σε επιχειρήσεις να ανεβάσουν τον κατάλογο των προϊόντων τους και να διαχειρίζονται τις παραγγελίες που δέχονται στα προϊόντα τους.
Λειτουργικότητες
Σε αυτή την ενότητα θα περιγράψουμε τις λειτουργικότητες της εφαρμογής ανά οθόνη.
Οθόνες:
●	Εγγραφή Χρήστη
●	Σύνδεση Χρήστη
●	Onboarding του Χρήστη
●	Προφίλ Χρήστη
●	Αρχική σελίδα αναζήτησης
●	Σελίδα αποτελεσμάτων αναζήτησης
●	Σελίδα προϊόντων συγκεκριμένης κατηγορίας
●	Σελίδα συγκεκριμένου προϊόντος
●	Δημιουργία Επιχείρησης
●	Προφίλ Επιχείρησης
●	Προσθήκη προϊόντων στο καλάθι
●	Προβολή καλαθιού
●	Δημιουργία παραγγελίας
